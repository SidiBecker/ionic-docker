# Ionic Docker
A environment image to be used with Gitlab CI

### Inspired by:
- https://github.com/marcoturi/ionic-docker

### Features
- Node
- Npm or Yarn
- Ionic
- Cordova
- Android sdk and build tools
- Ruby
- Scss-lint supports
- Python3 and PIL library
